# Java Learning & Practice

A collection of Java files for practice, learning, and testing algorithms.

## Directory Structure

*   `src/completed/`: Finished implementations (e.g., mathematical equations).
*   `src/live/`: Main execution entry points.
*   `src/practice/`: Learning exercises and experimental logic.
*   `src/random/`: Miscellaneous tests, including recursion examples.
*   `src/utils/`: Base classes and shared utilities.
